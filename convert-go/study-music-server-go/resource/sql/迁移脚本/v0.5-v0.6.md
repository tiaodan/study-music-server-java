# v0.5 -> v0.6 迁移脚本

## 本次变更功能

重命名字段，使命名更清晰：

1. spider_url → spider_url_path（爬取链接的path部分）
2. full_name → full_name_singer（多歌手时存储，单人则为空）

## 执行方式

```bash
mysql -u root -ppassword tp_music < sql/迁移脚本/v0.5-v0.6.sql
```

## 回滚方案

```sql
ALTER TABLE song CHANGE COLUMN spider_url_path spider_url VARCHAR(500);
ALTER TABLE song CHANGE COLUMN full_name_singer full_name VARCHAR(255);
```
