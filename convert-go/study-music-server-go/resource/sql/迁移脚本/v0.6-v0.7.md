# v0.6 -> v0.7 迁移脚本

## 本次变更功能

重命名字段：

1. nas_url → nas_url_path（明确是路径）
2. spider_url_path → spider_url（改为存完整URL，带http头）

## 执行方式

```bash
mysql -u root -ppassword tp_music < sql/迁移脚本/v0.6-v0.7.sql
```

## 回滚方案

```sql
ALTER TABLE song CHANGE COLUMN nas_url_path nas_url VARCHAR(255);
ALTER TABLE song CHANGE COLUMN spider_url spider_url_path VARCHAR(500);
```
